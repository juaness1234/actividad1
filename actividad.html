Cuestionario sobre Métodos Numéricos

1. ¿Cuál de los siguientes métodos se usa para encontrar raíces de ecuaciones?

2. ¿Qué técnica numérica se basa en la aproximación de derivadas?

3. ¿Cuál es un método iterativo para resolver sistemas de ecuaciones lineales?

4. ¿Cuál de estos métodos numéricos se usa en interpolación?

5. ¿Qué método numérico es útil para resolver ecuaciones diferenciales?






Cuestionario: Verdadero o Falso

1. El método de Bisección es un método numérico para resolver ecuaciones diferenciales.

2. El método de Gauss-Seidel es un método iterativo para resolver sistemas de ecuaciones lineales.

3. El método de Newton-Raphson es un método para interpolación de datos.

4. La interpolación de Lagrange es un método para encontrar una función que pase por un conjunto de puntos.

5. El método de Euler se usa para resolver ecuaciones diferenciales ordinarias.
